# 四柱八字排盘算命引擎

基于**子平法**·**高氏日柱公式**·**刘文元《四柱命理正源》**的纯 Python 八字排盘引擎。

## 功能

✅ **四柱排盘** — 年柱、月柱(节气分界)、日柱(高氏公式)、时柱
✅ **真太阳时校正** — 内置40+城市经纬度数据库
✅ **农历↔公历转换** — 支持农历输入(`--lunar` 参数)
✅ **五行量化** — 四柱五行百分比统计分析
✅ **十神** — 天干十神 + 藏干十神(本气/中气/余气)
✅ **十二长生** — 日干在各柱的十二长生状态
✅ **空亡** — 日柱空亡计算
✅ **神煞** — 天乙贵人、驿马、桃花、羊刃、禄神、华盖、将星、魁罡
✅ **身强身弱** — 基于月令、生扶、克泄耗综合判断
✅ **格局判定** — 正官/七杀/正印/偏印/正财/偏财/食神/伤官/建禄/月劫
✅ **用神体系** — 用神/喜神/忌神一生不变
✅ **大运排盘** — 阳男阴女顺排/阴男阳女逆排，每十年一运
✅ **流年分析** — 附带六冲六合关系
✅ **藏干** — 正确顺序(丑:己癸辛 非己辛癸)

## 使用方法

### 基本调用

```bash
python3 scripts/bazi_engine.py YEAR MONTH DAY HOUR MINUTE GENDER
```

### 示例

```bash
# 男命 1978年10月1日 18:30
python3 scripts/bazi_engine.py 1978 10 1 18 30 male

# 女命 1982年6月23日 17:10
python3 scripts/bazi_engine.py 1982 6 23 17 10 female

# 农历输入
python3 scripts/bazi_engine.py --lunar 1978 8 30 18 30 male

# 指定出生城市
python3 scripts/bazi_engine.py 1982 6 23 17 10 female 北京

# 直接指定经纬度
python3 scripts/bazi_engine.py 1982 6 23 17 10 female --lng 116.4 --lat 39.9
```

### 输出

输出为 JSON 格式，包含完整的四柱信息、五行、十神、大运、流年、神煞等。

## 核心算法

### 日柱 — 高氏日柱公式

基准日：1900年1月1日 = **甲戌日**(gan=0, zhi=10)

```python
base_date = datetime(1900, 1, 1)
days_diff = (target_date - base_date).days
gan_idx = (0 + days_diff) % 10
zhi_idx = (10 + days_diff) % 12
```

### 月柱 — 节气分界 + 五虎遁

以节气(节)为月柱分界线，非农历初一:
- 寅月: 立春(2/4)
- 卯月: 惊蛰(3/6)
- 辰月: 清明(4/5)
- 巳月: 立夏(5/6)
- 午月: 芒种(6/6)
- 未月: 小暑(7/7)
- 申月: 立秋(8/7)
- 酉月: 白露(9/8)
- 戌月: 寒露(10/8)
- 亥月: 立冬(11/7)
- 子月: 大雪(12/7)
- 丑月: 小寒(1/6)

五虎遁数组从寅月开始(索引0=寅月)。

### 时柱 — 五鼠遁

四柱以月令为纲，日干为体，时干为归。

**戊午 辛酉 丙申 丁酉** (1978-10-01 18:30 男):

## 注意事项

- 时辰以真太阳时为准(非北京/标准时区时间)
- 日柱以**子时23:00**为换日分界
- 正月天干用五虎遁，时干用五鼠遁
- 用神一生不变(刘文元《正源》核心原则)
- 月支是衡量全局干支旺衰的唯一标准
